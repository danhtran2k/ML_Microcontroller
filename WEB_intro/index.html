APPLICATION OF MACHINE LEARNING MODE TO MICROCONTROLLERS - AUTOMATION OF IOT EDGE DEVICES

The topic uses a machine learning model to realize the automatic process of lights and fans through training data on temperature, humidity, light and whether or not people are in the room.

System model

Node sensor

Node temperature, humidity, light level, dectect people

Gateway

Build gateway with esp32 microcontroller, applycation model machine learning on gateway control lamp & fan.

Node switch

Device turn on, turn off lamp and fan

Server

Server node js connects to gateway and mobile app

App mobile

Using reactnative platform, connect to the server to monitor and control sensors and devices

Protocol

Use RF radio protocol to transmit information between gateway and device node. MQTT protocol for transmitting and receiving information between gateway and server. And use socket.io to communicate between mobile app and server.

Neural network in the model

Use layers with number of buttons tested: Input layer 100 units, two layers Hidden layer 64 units and 16 units, Output layer with 2 units

Step by step buld model ML on microcontroller

Conclude

Edge computing on gateway

Edge computing is a technology that brings computing to the end devices. In EDGE-C the computations performed at the centre of the network which is the cloud is moved to end devices. This reduces the complexity and overhead of processing large computation in the cloud. This also decreases the network usage and enables fast responses which result in better transfer rate.

App mobile

App monitoring temperature, humidity, light level, count people and control lamp & fan
